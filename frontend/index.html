How to Set Up Test Tasks for Demo

To demo the reminder system, upload a meeting with tasks that have deadlines set to "today" or "tomorrow":

Example Meeting Transcript: 
[Sprint Standup - Engineering]

Sarah: I'll finish the API documentation by today.
Owner: Sarah
Deadline: today

Tom: I need to review the PR by tomorrow.
Owner: Tom
Deadline: tomorrow

Alice: Deploy to staging - blocked on DB migration.
Owner: Alice
Deadline: Friday
Status: blocked

Supported Deadline Formats:

  • today, tomorrow
  • Day of week: Friday, Monday
  • ISO date: 2026-02-21
  • Relative: in 3 days

Architecture
K2-Think-V2 → Reasoning: extracts tasks/decisions/risks, cross-meeting analysis
K2-V2-Instruct → Q&A: answers questions with department memory context
OpenClaw → Execution: task board, file writes, Slack/Notion integrations
Notion → Live task board synced on every meeting upload
Chroma + SQLite → Department memory: semantic search + structured data

cp openclaw/openclaw.json ~/.openclaw/openclaw.json
